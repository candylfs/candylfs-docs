# 課金体系

## 基本料金

- **1GBのデータパック**: 月額3円
- 容量は任意で設定可能

## 請求サイクル

- 課金開始日を起点として1ヶ月ごとに自動請求
- 最初の1ヶ月分は契約時に前払い(サブスクリプション方式)

### 例
1月2日に契約した場合:
- 1月2日: 初月分(1月2日〜2月2日)を支払い
- 2月2日: 次月分(2月2日〜3月2日)を自動請求
- 以降、毎月2日に自動請求

## 解約について

- **解約タイミング**: いつでも解約可能
- **返金**: 支払い済みの料金は返金されません

### 例
- 1月2日に契約、1月3日に解約した場合
  - 支払い済みの期間(2月2日まで)は引き続き有効
  - 返金なし

## 容量変更

契約期間中に容量を自由に変更できます。

### 容量を増やした場合

差額分が即座に請求されます。

**例**: 1月2日に50GB(150円)で契約した場合
- **1月3日に100GB(300円)に変更**
  - 差額: 300円 - 150円 = **150円を即座に請求**

### 容量を減らした場合

差額分が日割り計算でクレジットされ、次回以降の支払いに自動適用されます。

**例**: 1月2日に50GB(150円)で契約した場合
- **1月3日に25GB(75円)に変更**
  - 差額: 150円 - 75円 = 75円
  - 残り日数分が日割りでクレジット化
  - 次回請求時に自動的に割引として適用

## クレジットの適用範囲

- **重要**: 請求はテナントごとに発生します
- クレジットは**同一テナント内でのみ**利用可能
- 異なるテナント間でクレジットを引き継ぐことはできません

### 例
- テナントAで発生したクレジット → テナントAの次回請求にのみ使用可能
- テナントBには適用されません
