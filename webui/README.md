# WebUIの使い方

## テナントの作成

Create Tenantボタンより作成します。テナントごとにリポジトリや課金を管理します。

## リポジトリの作成、削除

テナント作成後、Repositoriesタブ→Create Repositoryで作成します。
リポジトリを削除した場合、自動で容量が解放されます。

## トークンの作成、削除

Tokensタブ→Create Tokenで作成します。
- トークン名
- スコープ(テナント内全体 or 特定のリポジトリ)
- パーミッション(read only or read and write)
- 期限(1日~365日)

を設定できます。

## 管理メンバーの招待、削除

Membersタブ→Invite Memberで作成します。admin権限を持つユーザーのみ招待ができます。idは右上のアイコンボタンを押すことで確認できます。

**admin権限はテナント全てへの権限を持っています。慎重に扱ってください、**

### Member権限

| 操作 | 説明 |
|------|------|
| テナント情報取得 | テナントの詳細情報を閲覧 |
| リポジトリ一覧取得 | テナント内のリポジトリ一覧を閲覧 |
| トークン作成 | LFSアクセストークンを作成 |
| トークン一覧取得 | 既存のトークン一覧を閲覧 |
| トークン削除 | トークンを削除 |
| 使用量取得 | ストレージ使用量を確認 |
| メンバー一覧取得 | テナントのメンバー一覧を閲覧 |
| GitHub設定取得 | GitHub連携設定を閲覧 |
| GitHub設定変更 | GitHub連携設定を変更 |
| GitHub設定削除 | GitHub連携設定を削除 |
| サブスクリプション情報取得 | 課金状況を閲覧 |

### Admin専用

| 操作 | 説明 |
|------|------|
| テナント削除 | テナント自体を削除 |
| リポジトリ作成 | 新規リポジトリを作成 |
| リポジトリ削除 | リポジトリを削除 |
| メンバー招待 | 新しいメンバーを招待 |
| メンバー削除 | メンバーをテナントから削除 |
| 課金セッション作成 | ストレージ購入のCheckout作成 |
| 課金ポータル | Stripeポータルへのアクセス |

## GitHub Integrationと削除

[cli](../cli/)と連携し、GitHub認証でlfsトークンを発行できます。

### Organizations
指定した組織に所属しているGitHubアカウントに対して自動でlfsトークンを発行できます。スコープは他のトークンと同様に設定できます。

**原則個人設定の方が組織設定より先に評価されます。個人アカウントで特定のリポジトリだけを許可していた場合、組織のスコープが全て許可になっていても特定のリポジトリだけにアクセス出来ます。逆に個人アカウントで全てのスコープを持っていた場合、組織のスコープが特定のリポジトリだけを許可していた場合でも全てのリポジトリにアクセス出来ます。**

### Individual Users
個人アカウントに対して自動でlfsトークンを発行できます。スコープは他のトークンと同様に設定できます。
